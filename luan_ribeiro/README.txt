descrição do sistema: um e-commerce de dispositivos móveis apple e samsung.

nas tela inicial temos os banner promocionais (carousel), o grid dos produtos da loja, o rodape, uma barra de navegação voltada para login, cadastro e página inicial. E navBar outra voltada para os produtos (com algumas opções de filtros).

nas telas do administrador do sistema temos uma tabela para os produtos da loja (com opções de edição e exclusão) e outra para os usuários banidos do sistema por algum motivo. Também tem uma navBar para facilitar a pesquisa em ambas as tabelas.

na tela de login, o sistema permite logar como usuário ou administrador. Também dá a opçao de ir para a tela de cadastro, caso o usuário não possua conta no sistema.

na tela de cadastro o sistema dá o feedback ao usuário ao cadastrar a conta. Também dá a opçao de ir para a tela de login, caso o usuário já possua conta no sistema.

na tela inicial, ao clicar nos produtos (configurado apenas para o Iphone 11 e Galaxy S23 UlTRA) leva para outra tela onde apresenta mais fotos para o usuário e uma melhor descrição do produto.
