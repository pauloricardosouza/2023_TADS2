O Projeto é uma Forma de Conectar usuarios de um game, onde estes poderam efetuar compras de 'contas' de forma simplificada e segura, sendo dispostas varias informações sobre as contas e preços.

1. Cadastro.html
Projeto: Página de cadastro para um site chamado "Dark Place Of Space".
Design e Layout: Utilização do framework Bootstrap para o estilo e responsividade. A página possui um cabeçalho com navegação, uma seção de cadastro com campos como Nome, E-mail, Login, Senha e Confirmar Senha, além de um botão de cadastro. O rodapé contém informações de autoria.
2. Carrinho.html
Projeto: Página de carrinho de compras para o mesmo site.
Design e Layout: Também utiliza Bootstrap. O cabeçalho e navegação são semelhantes a outras páginas. A seção principal exibe uma lista de itens no carrinho, o total e botões para comprar e voltar. O rodapé é consistente com as outras páginas.
3. Home.html
Projeto: Página inicial do site "Dark Place Of Space".
Design e Layout: Bootstrap é empregado novamente. O cabeçalho tem uma navegação destacando a página atual. Há seções de destaque, vídeos do YouTube e links de conexão. O rodapé fornece informações de autoria. O script ativo destaca o link da página atual na navegação.
4. Login.html
Projeto: Página de login para o site.
Design e Layout: O layout é semelhante às outras páginas com Bootstrap. O cabeçalho contém navegação, e a seção de login inclui campos para usuário e senha, além de um botão para entrar. O rodapé tem informações de autoria. Um script valida o login com informações simuladas.
5. Produto.html
Projeto: Página de exibição de produtos do site.
Design e Layout: Usa Bootstrap para o layout. O cabeçalho possui navegação destacando a página atual. A seção principal exibe produtos em um carrossel com informações e botão "Mais Informações". O rodapé é semelhante aos outros, e um script controla o carrossel e as interações.
6. MaisInformacoes.html
Projeto: Página com mais informações sobre os produtos.
Design e Layout: Usa Bootstrap e mantém consistência com as outras páginas. Possui navegação, informações detalhadas sobre os produtos e um botão "Voltar". O rodapé é semelhante. Um script provavelmente controla a navegação.

DETALHAMENTO DO USO DOS DESIGNS
Consistência: Todas as páginas mantêm uma consistência visual com o uso do Bootstrap e a estruturação semelhante de cabeçalho, navegação, seções principais e rodapé.

Responsividade: A escolha do Bootstrap contribui para a responsividade, adaptando-se a diferentes tamanhos de tela.

Interatividade: Os scripts são utilizados para fornecer interatividade, como validação de login, controle de carrossel e navegação destacada.

Informações de Autoria: Cada página possui informações de autoria no rodapé.
