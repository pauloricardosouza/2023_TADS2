Projeto - Loja de calçados 

Alisson Ribeiro Carneiro

	A intenção do projeto foi criar uma loja online de venda de calçados inspirada em lojas já existentes como PUMA (https://br.puma.com/) e Nike (nike.com.br).
Utilizei estas lojas apenas como referência para alguns itens, como um banner na página inicial com ofertas de Black Friday, divisão de categorias entre "Black Friday", "Masculino", "Feminino" e "Infantil".
	Utilizei uma NavBar fixa no topo, onde estão as divisões de categorias mencionadas acima e também os botões de login e carrinho de compras. A página do carrinho de compras não foi editada, ela é apenas um HTML vazio.
	Utilizei de um Carousel para criar um banner com duas imagens logo no inicio da página. Criei estas imagens no Canva, assim como a logo da loja COLOR SNEAKERS e o ícone exibido na guia do navegador. Tentei respeitar a paleta de cores, utilizando do contraste criado pelo fundo preto com as cores vivas da logo.
	Utilizei novamente o recurso Carousel para criar uma exibição de 4 imagens que levam a categorias de "Corrida", "Futebol", "Especiais" e "Dia a Dia", porém não organizei os itens dentro de cada uma, elas apenas levam a uma página comum.
	Em seguida utilizei uma Grid com 6 elementos para exibir os itens mais vendidos da loja e após, um Footer com dados aleatórios apenas para efeito visual.
	Dentro da guia "Black Friday", fiz uso de um alerta do tipo "alert-danger" para exibir uma mensagem sobre o fim da promoção de Black Friday. As demais guias apenas exibem itens de sua categoria. Ao clicar em algum item, será levado a uma página comum de detalhamento do item, sem muita formatação.
	Na página de login usei como inspiração a página de login da PUMA, onde o formulário de login é exibido ao lado do formulário de cadastro. Usei apenas placeholders para identificar os campos.
