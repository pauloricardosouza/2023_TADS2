Este site foi feito com o intuito de dar uma noção do que é o santos futebol clube que é o clube que eu torço.
escolhi as cores do projeto baseado nas cores tradicionais que o clube usa.
também criei a logo inspirada no time para dar uma identidade visual melhor.
O site tem muitos cards que mostram alguns jogadores, idolos e simbolos do time, adicionei tambem uma loja tematica com uniformes do clube.
usei pouco java script por falta de conhecimento sobre o assunto.
peguei algumas coisas do bootstrap e outras de outras atividades feitas em sala então tem muita coisa parecida com o que ja fizemos em aula.
tentei fazer uma barra que fosse compativel tambem com celular porem nao inclui coisas como carrosel ou cards responsivos.
