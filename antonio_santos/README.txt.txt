# Projeto CpUStore

## Descrição do Projeto
Este projeto consiste em uma loja virtual chamada CpUStore, onde usuários podem visualizar produtos, realizar login e acessar o sistema. O sistema é voltado para a disciplina de Tecnologias Web.

## Estrutura de Diretórios
- `img`: Contém as imagens utilizadas no projeto.
- `index.html`: Página inicial da loja.
- `produto.html`: Página de cadastro de produtos.
- `formLogin.html`: Página de login.
- `realizarCompra.html`: Página para realizar a compra (necessita implementação futura).
- `README.txt`: Este arquivo.

## Escolhas de Design e Layout
- **Bootstrap**: Utilizei o framework Bootstrap para garantir um design responsivo e uma aparência moderna.
- **Navbar**: Adicionei uma barra de navegação (navbar) para facilitar a navegação entre as páginas.
- **Cores**: Escolhi uma paleta de cores azul escuro para o cabeçalho e rodapé, proporcionando uma identidade visual consistente.
- **Fonte**: Utilizei a fonte "Roboto" para garantir uma leitura clara e moderna.
- **Carousel**: Implementei um carrossel na página principal para exibir produtos de forma atrativa.

## Modificações Recentes
